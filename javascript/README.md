# javascript 학습

## **반복문**

> 기본 for문
```javascript
    for(i = 0; i<flowers.length; i++)
        console.log(flowers[i]);
```
> for...in문
```javascript
    for(i in flowers)
        console.log(i, flowers[i]);
```
> for..of문
```javascript
    for(item of flowers)
        console.log(item);
```

> forEach문
```javascript
    flowers.forEach((x) => console.log(x));
```
